buka cmd, arahkan directori ke folder server, jalankan server dengan perintah mvn spring-boot:run
Simpan folder client di webserver local (yang di testing menggunakan XAMPP)
Cut folder client paste kedalam folder c:/xampp/htdocs
Nyalakan Xampp apache, ketikan ke addres bar browser localhost/client
connect ke websocket.
(SCHEDULLER EXAMPLE) coba push notif setiap 5 detik atau 10 detik
(CLIENT SENT) Client sent request ke server, server mengembalikan response ke client.
